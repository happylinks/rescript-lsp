# ReScript LSP Plugin for Claude Code

A Claude Code plugin that provides ReScript language server support for code intelligence.

## Prerequisites

Install the ReScript language server globally:

```bash
npm install -g @rescript/language-server
```

## Installation

Add this plugin to Claude Code:

```bash
claude /plugin add https://github.com/happylinks/rescript-lsp
```

Then restart Claude Code.

## Features

- Code intelligence for `.res` and `.resi` files
- Go to definition
- Find references
- Hover information
- Real-time diagnostics
